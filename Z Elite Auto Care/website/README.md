# Z Elite Auto Care Website

Premium mobile car detailing and garage services website for Darwin, NT.

## Files

- `index.html` - Main website file
- `style.css` - All styles and animations
- `script.js` - Interactive functionality
- `images/` - Image assets (21 files)

## Features

- Glassmorphism navbar with scroll effect
- Word-by-word hero text animation with 3D flip
- Button ripple effects
- Parallax hero background
- Enhanced service card hover effects
- Page loader animation
- Scroll-to-top button
- Mobile responsive design
- Web3Forms booking integration
